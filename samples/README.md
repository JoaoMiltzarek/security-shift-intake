# Samples

Committed **synthetic** example renders (PNG) produced by `make gen-pdfs`, kept so
the Tier B output can be eyeballed without regenerating it.

Policy: only synthetic, code-generated images belong here. The pre-commit guard
(`scripts/check_real_data.py`) blocks image/PDF files everywhere else — this folder
is the single, narrow exemption (PNG/JPG only). **Never** put a real scan here.
