# Samples

Committed **synthetic** example renders produced by repository fixtures, kept so the
Document AI output and review UI can be inspected without any real document.

Policy: only synthetic, reproducible media belongs here. The pre-commit guard
([scripts/check_real_data.py](../scripts/check_real_data.py)) uses a filename allowlist,
not a blanket directory exception. PNG/JPG names generated by the fixture pipeline are
allowed; the only animated exception is exactly `samples/cockpit_demo.gif`. Every other
GIF, image or document name remains blocked, including nested files and files under
`assets/`. **Never put a real scan here.**
